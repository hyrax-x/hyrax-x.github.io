Soft & Hard

  • Project
  • Art
  • Comic

Description

This is a semester's work for the Visual Narration class. The task is to create a comic based on an opposite word: soft & Hard. In my story, the cat represent soft, the vacuum represent hard.

The comic

using ClipStudioPaint, I draw the comic in grayscale and apply masked gradient to have a cool color effect.
