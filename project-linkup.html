Description

  • This is an app about learning language by connecting with people and learn language in a natural way.
  • The app will be using matching algorithm to match people with the same hobby and the desire to learn the language from the other person. When the user found a match, the app will provide fun activity and lessons to increase engagement and create opportunities for the user to learn language from another.
