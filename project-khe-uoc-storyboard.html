Khe Uoc Storyboard

  • Project
  • Animation

  • Art

Description

This is a semester's work for the Visual Narration class. The task is to create a storyboard based on a poem or song lyrics. I chose Khe Uoc by The Flob because the song interested me and has no official music video.

PDF Link

-> Khe Uoc Storyboard PDF <-

Script and sketch

I analyzed the song's chorus lyrics and, based on that analysis, developed an initial storyboard concept.

Final illustration

Using ClipStudioPaint, I sketch the initial outlines, refining ideas as I go. Once satisfied with the sketch, I develop and bring my idea to life by adding layers, colors, and textures to create a finished digital artwork.
